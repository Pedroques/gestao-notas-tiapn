# Nome do Projeto: ProControl - Dodora Modas
Este projeto é um sistema de gestão desenvolvido como parte da disciplina Trabalho Interdisciplinar: Aplicações para Processos de Negócios. 
O objetivo central é otimizar o fluxo operacional de um comércio local, automatizando o controle de estoque, o cadastro de colaboradores e clientes, e o registro de transações de vendas.

## Funcionalidades Principais
- O sistema foi modelado para atender aos seguintes pilares de gestão:
- Gestão de Estoque: Entrada e saída de produtos, controle de níveis mínimos e visualização de itens em falta.
- Gestão de Vendedores: Cadastro de funcionários, controle de acesso e histórico de vendas por vendedor.
- Gestão de Clientes: Cadastro, atualização de contatos e histórico de compras individuais.
- Registro de Vendas: Emissão de pedidos, cálculo de totais e associação entre produtos, vendedores e clientes.
- Relatórios: Visualização das métricas da empresa, com parâmetros estabelecidos pelos usuários.
